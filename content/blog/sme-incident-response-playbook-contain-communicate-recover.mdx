# SME Incident Response Playbook

Ce guide explique comment une PME peut réagir efficacement à un incident de sécurité.

## 1. Contenir
- Isolez les systèmes affectés
- Changez les mots de passe compromis

## 2. Communiquer
- Prévenez les parties prenantes
- Gardez une trace écrite de toutes les actions

## 3. Récupérer
- Restaurez les sauvegardes
- Analysez la cause racine

> Un plan d'action clair limite l'impact d'une crise.
